Fuse is the compiler oracle for .NET coding agents. Its thesis is that the compilation is
the source of truth, so it answers an agent's edit-time questions from the compiler rather
than from a guess: it typechecks a proposed edit before the agent writes it, computes the
blast radius of a change, stages compiler-executed refactors as diffs that compile or are
not handed over, and resolves what the code actually runs (which implementation is injected
for an interface, which handler processes a request, which endpoint serves a route, what a
git diff impacts). Every answer carries a grade, and Fuse abstains rather than guess when it
cannot answer at compiler grade. Scoped, reduced context and ranked retrieval are the
supporting machinery that feed those answers. It ships as a Model Context Protocol server
(`fuse mcp serve`) for an agent, and as a .NET global tool (`fuse`) for the terminal and CI.

These docs are organized so you can read top to bottom to learn, or jump to a
reference page for exact detail.
