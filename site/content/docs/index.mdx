---

Fuse gives an existing coding agent a local connection to the .NET compiler and a typed
graph of application wiring. Through Model Context Protocol (MCP), Claude Code, Cursor,
GitHub Copilot, and other MCP clients can check a proposed edit, inspect typed impact,
resolve dependency injection and framework wiring, and stage compiler-verified refactors.

Fuse runs on the developer's machine. It requires no hosted model and downloads no
embedding model. Compiler answers report their evidence grade, and Fuse abstains when its
available compiler path cannot support a verdict. Checks evaluate in-memory proposals
without writing the working tree. Ranked retrieval and structural reduction supply compact
context to these operations.

Install the .NET tool, then register it with your agent:

```bash
dotnet tool install -g Fuse
fuse mcp install --rules
```

These docs are organized so you can read top to bottom to learn, or jump to a
reference page for exact detail.
