Fuse is a local .NET tool with a persistent semantic index and pre-write compiler
verification for coding agents. Install and connect from a project directory:

```bash
dotnet tool install -g Fuse
fuse mcp install --rules
```

Reload your MCP client, then ask:

```text
Resolve IOrderService to its implementation, then check the proposed OrderService.cs
content with fuse_check before writing the file.
```

Fuse runs on your machine. It stores a warm index in `.fuse/fuse.db`, walks typed
wiring edges when the project loads through Roslyn, and returns compiler diagnostics for
proposed single-file edits without changing the working tree.
