title: Fuse documentation
description: Install Fuse, connect your coding agent, and choose the workflow for checking edits, tracing .NET wiring, reviewing changes, or reducing context.
---

Give your existing coding agent local compiler evidence and a typed graph of .NET
application wiring:

```bash
dotnet tool install -g Fuse
fuse mcp install --rules
```

Restart your MCP client, then ask it to call `fuse_workspace` with `action=status`.
