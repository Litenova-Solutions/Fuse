Catch compiler errors before your coding agent writes a proposed C# edit:

```bash
dotnet tool install -g Fuse
fuse mcp install --rules
```

Reload your coding agent, then ask:

```text
Check the next edit with fuse_check before writing it.
```

Fuse runs against the repository on your machine. It also traces dependency injection,
requests, routes, and configuration to the code that handles them.
