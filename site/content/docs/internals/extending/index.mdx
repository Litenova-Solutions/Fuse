title: Extending Fuse
description: Add languages, templates, format reducers, and pattern detectors through the capability model.
---

Fuse keeps language and format behavior in plugins resolved by file extension, so extending it means adding a capability rather than changing the core pipeline. This section is for contributors. Read [Capability and Plugin Model](/docs/internals/capability-model) first for the design these guides build on.

## Pages

- [Adding a Language Plugin](/docs/internals/extending/language-plugin): register reduction, skeleton, dependency, and other capabilities for a language.
- [Adding a Template](/docs/internals/extending/template): declare default extensions and exclusions for a project type.
- [Adding a Format Reducer](/docs/internals/extending/format-reducer): reduce a non-language file type.
- [Adding a Pattern Detector](/docs/internals/extending/pattern-detector): detect a convention and contribute to the pattern summary.
